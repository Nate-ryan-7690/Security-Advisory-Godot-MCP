Security Policy

This repository publicly documents a security advisory for a third-party project (Godot-MCP). It is published for defensive purposes so affected users 
can identify and mitigate the issue.

All testing was performed by the researcher in isolated virtual machine environments under the researcher's sole control. No third-party or production 
systems were accessed at any point. Working exploit code is not included in this repository; the advisory provides only the technical detail needed for 
defenders and security tooling to understand and mitigate the vulnerability.
